Trabajo Final de transferencia de archivos para Taller de programación 3, de la carrera AUS.

Para compilar:
javac -d bin src/com/fileshare/*.java


Para ejecutar la aplicacion:
java -cp bin com.fileshare.Main


Casos probados:
 Luego de compilar, abrir dos terminales simulando dos computadoras.
 Ingresar el usuario en ambas terminales, una con enviar archivo y otra para recibir
 Ingresar la ruta del archivo, pwd/prueba.txt

 Si todo salió bien, el archivo va a aparecer en la carpeta Downloads.


Ademas se probó con un celular android y la aplicación Termux, tanto para enviar y recibir archivos.

Hecho por Fabrizio, con la teoría y práctica vista en Taller de programación 3 por Luciano Diamand.
